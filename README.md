פרויקט: IDFK – Marvel Community Hub & Real-Time Management
טכנולוגיות: ASP.NET Core (MVC/Razor) | SignalR | EF Core | SQL Server/SQLite | Bootstrap 5 | JavaScript (ES6+)
תיאור כללי:
פיתוח פלטפורמת קהילה אינטראקטיבית מקיפה למעריצי MARVEL, המשלבת מערכת ניהול תוכן (CMS), ניהול משתמשים מתקדם וחוויית משתמש דינמית בזמן אמת. הפרויקט מדגים שילוב של טכנולוגיות צד-שרת חזקות עם ממשק קצה (Frontend) עשיר ומותאם אישית.
יכולות מרכזיות (Key Features):
מערכת צ'אט קהילתי בזמן אמת (SignalR):
פיתוח צ'אט חי המוטמע בדפי הסרטים ומאפשר אינטראקציה מיידית ללא רענון דף.
ניהול תוכן בזמן אמת: פיתוח יכולת מחיקת הודעות לצד השרת (Admin Only) עם עדכון ויזואלי מיידי לכל המשתמשים המחוברים.
סנכרון מלא בין זהות המשתמש, תמונת הפרופיל והודעות הצ'אט במסד הנתונים.
חוויית משתמש אינטראקטיבית (Advanced UI/UX):
Dynamic Theme Engine: פיתוח רכיב החלפת ערכות נושא (Themes) מותאם אישית המבוסס על דמויות מארוול (Iron Man vs. Captain America), הכולל שינוי משתני CSS (Variables) בזמן אמת ושמירת העדפות המשתמש ב-LocalStorage.
S.H.I.E.L.D Mode: הוספת מצב ניטרלי (Dark Mode) המוטמע ב-Navbar לאיזון ויזואלי.
ניהול תוכן ונתונים (Content & Data):
גלריית סרטים דינמית הכוללת דפי פרטים עם טריילרים (YouTube Embed API).
מנגנון העלאת קבצים (File Upload) מתקדם עבור פוסטרים ותמונות פרופיל אישיות.
עבודה עם Entity Framework Core בגישת Code-First וביצוע שאילתות LINQ מורכבות לשליפת מידע.
אבטחה וניהול משתמשים (Security & Auth):
מימוש Cookie-based Authentication מאובטח עם הצפנת סיסמאות.
ניהול הרשאות קפדני (Role-based Authorization) המפריד בין יכולות צפייה של משתמש לבין יכולות ניהול (Admin Panel).
דגשים טכניים וארכיטקטורה:
Clean Code: שימוש ב-Dependency Injection (DI) להזרקת שירותים ומסדי נתונים.
Validation: וולידציה כפולה (צד לקוח וצד שרת) בטפסי הוספת תוכן ורישום משתמשים.
Responsive Design: התאמה מלאה לכל סוגי המסכים באמצעות Bootstrap 5 ו-Media Queries מותאמים.
